# Laravel Framework 11.34.2

## Preoject setup

### Install composer
```sh
composer install
```

### Update .env file for database connection
```sh
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=test_blog_abdullah
DB_USERNAME=root
DB_PASSWORD=P@55word
```

### Database migrations
```sh
php artisan migrate
```
